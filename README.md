# ati-videocard-showcase
An obscure program/showcase of ATI's selection of videocards from the early 2000s.

## Viewing
You'll need to view this in a browser that supports ActiveX, due to it's age. Internet Explorer *should* work - but you  might need to use compatibility view or use an older version. I'd try it in IE 5 or 6 for the best results.

## Setup
In order to view it, you'll need to install the ActiveX plugin. Run `cult3d_ie.exe` and then open `index.html` in your browser (see above for compatibility).
